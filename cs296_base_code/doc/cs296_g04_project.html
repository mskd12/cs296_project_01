Introduction

This project was done as a part of the Software Systems Course (CS296). We had to, at the start of the course propose a simulation of a complex machine for the project. This is the link for our earlier project proposal. This simulation was done using the physics engine Box2D. The entire project was coded up in C++.


Links to the Profiling Data Analysis

The simulation was profiled with perf and the data was analysed. This profiling was done by disabling the graphics and the GUI part of the simulation to get the data for the actual simulation.
